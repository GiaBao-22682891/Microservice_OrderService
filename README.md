## 👥 Thành viên nhóm & Phân công nhiệm vụ
Dự án được phát triển theo kiến trúc **MicroService*, phân chia như sau:

* **Trần Phúc Hưng (Nhóm trưởng):** `UserService` 
* **Tô Phan Gia Bảo:** `OrderService` 
* **Trần Thành Đạt:** `FoodService` 
* **Lê Ngọc Dung:** `Payment` + `Notification Service` 
* **Mai Lưu Hữu Vinh:** `Front-end` (FE)
